## ğŸ¨ Using the Design System

### Copy a Color

1. Navigate to Colors page
2. Click any color swatch
3. HEX code is copied to clipboard
4. Paste into your code

### Copy a Component

1. Navigate to Components page
2. Click to expand a component
3. Copy the import statement
4. Copy the component code
5. Paste into your project

### Copy an Icon

1. Navigate to Icons page
2. Search for the icon you need
3. Click the icon
4. Import and usage code copied to clipboard
